  <action>
1. Export command (`pm export [path]`):
   - Read manifest.json from data/manifest.json
   - If path argument provided, copy manifest to that path
   - If no path, print manifest JSON to stdout
   - Exit with error if no manifest found

2. Overlaps command (`pm overlaps`):
   - Read manifest.json
   - Import and call findOverlaps() from src/lib/overlaps.ts (adjust import path as needed — may need to move findOverlaps to a shared location or use relative import)
   - Print table to console: capability name, project count, project names
   - Format as aligned columns for terminal readability
   - Exit with error if no manifest found
   - If no overlaps found, print "No capability overlaps detected."

Note: findOverlaps is in src/lib/overlaps.ts which imports from scanner/types.ts. Since cli.ts is in scanner/, the import path should work. If not, use a relative import path.
